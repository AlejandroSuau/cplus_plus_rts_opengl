# ♟️ RTS Game OpenGL

## 🔧 Technology Stack

**Compilation:**  
- C++20  
- CMake  
- Make  

**Framework & Libraries:**  
- OpenGL
- Glad

---

## 💻 Build & Run Instructions

### 🖥️ Windows

```bash
cmake -B build
cd build
cmake --build .
.\Debug\RTSGame.exe
```

### 🍎 MacOS / Linux

```bash
cmake -B build
cd build
make
./RTSGame
```
